# BAMAZON | SQL
A store Forefront
- It will take in orders from customers and deplete stock from the store's inventory


 ### Why use LIRI BOT? ###
 Use BAMAZON to get 
 * stock quantity left in store
 * order goods

 
 
 
 
 
 
 
 ### How to get started? ###
